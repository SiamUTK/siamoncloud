# Siam On Cloud

Production-ready React + Vite starter structure for an AI-powered travel technology platform.

## Tech Stack

- React
- Vite
- JavaScript
- Tailwind CSS
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
